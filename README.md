# React Based - Shows App
It's a react base show's website, where you can login, signin & book tickets for movies.
### Learned Things 
- React Component
- Redux
- state Managment
- Login/ Signin with localstorage
- useCallback,UseEffect, UseState
- Axios
### Used Npm package
- Ant Design
- Ant Icons
- Tailwind Css
- React Icons
- Redux-Toolkit
